Hypertext Preprocessor

What is PHP?

PHP stands for Hypertext Preprocessor. It's a server-side scripting language designed for dynamic web development.

Key Points:
  • ‚úì Runs on the server
  • ‚úì Open-source and free
  • ‚úì Embedded in HTML
  • ‚úì Cross-platform
  • ‚úì Interpreted (no compilation needed)

History of PHP

1994: Created by Rasmus Lerdorf
1995-1997: PHP 3.0 released
1998-2000: PHP 4 with Zend Engine
2004: PHP 5 introduced OOP
2015: PHP 7 with performance improvements
2020+: PHP 8 with JIT, union types, match expressions

Server-Side Processing

PHP runs on the server and generates HTML that is sent to the client's browser.

The client only sees the output, not the PHP code itself.

PHP Syntax Basics

<?php
// Print Hello World
echo "Hello, World!";

// Variables
$name = "Yogesh";
$age = 30;
echo "My name is $name and I am $age years old.";

// Conditional
if ($age > 18) {
echo "You are an adult.";
} else {
echo "You are a minor.";
}

// Loop
for ($i = 1; $i <= 5; $i++) {
echo "Number: $i<br>";
}
?>

Database Integration

PHP seamlessly integrates with various databases:

<?php

if (!$conn) {
die("Connection failed: " . mysqli_connect_error());
}
echo "Connected successfully";
?>
