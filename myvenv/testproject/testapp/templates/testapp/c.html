C

C Programming Language

A general-purpose, procedural programming language known for its speed, efficiency, and low-level memory access.

Foundation of Modern Programming

History of C

The C programming language was developed by Dennis Ritchie in 1972 at Bell Laboratories.

It was primarily designed for developing the UNIX operating system, which is still widely used today.

Why C is Important

  • ✓ It is very fast and efficient
  • ✓ It provides direct access to memory
  • ✓ It is used to build operating systems
  • ✓ Many modern languages are based on C
  • ✓ It helps understand how computers work internally

How C Works

C Source Code
→
Compiler (GCC)
→
Machine Code
→
CPU Execution

C code is compiled directly into machine code, making C programs extremely fast.

Basic C Program Structure

#include <stdio.h>

int main() {
printf("Hello, C Language");
return 0;
}

Every C program starts execution from the main() function.

Variables in C

Variables must be declared with a data type before use.

int age = 25;
float salary = 5000.50;
char grade = 'A';

Derived Data Types

  • Array

Conditional Statements

if (age >= 18) {
printf("Eligible to vote");
} else {
printf("Not eligible");
}

Loops in C

For Loop

for (int i = 1; i <= 5; i++) {
printf("%d", i);
}

While Loop

int i = 1;
while (i <= 5) {
printf("%d", i);
i++;
}

Functions in C

Functions help divide a program into smaller reusable parts.

int add(int a, int b) {
return a + b;
}

Pointers in C

Pointers store the address of variables and are a powerful feature of C.

Variable
x = 10
→
Pointer
*p = &x
int x = 10;
int *p = &x;

Structures in C

Structures are used to group different data types.

struct Student {
int id;
char name[50];
};

Uses of C Language

Operating Systems Embedded Systems Device Drivers Compilers Game Development

Advantages & Limitations

✓ Advantages

  • Very fast execution
  • Efficient memory management
  • Portable across platforms
  • Strong foundation for learning

✗ Limitations

  • No object-oriented features
  • Manual memory management
  • No built-in exception handling
  • Less secure than modern languages

Conclusion

C is a powerful and fundamental programming language that helps programmers understand core concepts of computing. Learning C provides a strong base for system programming and learning advanced languages.
