Java Programming Language

A high-level, object-oriented, class-based programming language designed to be platform-independent.

Write Once, Run Anywhere (WORA)

History of Java

Java was developed by James Gosling and his team at Sun Microsystems in 1995. It was originally called Oak.

Later, Sun Microsystems was acquired by Oracle Corporation, which now maintains Java.

Why Java is Popular

  • ✓ Is platform-independent
  • ✓ Is highly secure and reliable
  • ✓ Is used by large enterprises
  • ✓ Has strong community support
  • ✓ Is used in Android development

Java source code is compiled into bytecode which is then executed by the Java Virtual Machine (JVM).

Basic Java Syntax

class HelloWorld {
public static void main(String[] args) {
System.out.println("Hello, Java");
}
}

Java programs start execution from the main method.

Variables in Java

Java is a statically typed language, so variable types must be declared.

int age = 25;
String name = "Yogesh";

Data Types in Java

Primitive Data Types

  • int
  • float
  • double
  • char
  • boolean

Non-Primitive Data Types

  • String
  • Array
  • Class
  • Interface

Conditional Statements

if (age >= 18) {
System.out.println("Eligible to vote");
} else {
System.out.println("Not eligible");
}

Loops in Java

For Loop

for (int i = 1; i <= 5; i++) {
System.out.println(i);
}

While Loop

int i = 1;
while (i <= 5) {
System.out.println(i);
i++;
}

Object-Oriented Programming (OOP)

Java is fully object-oriented and supports:

Class & Object
Inheritance
Polymorphism
Abstraction
Encapsulation
class Student {

Exception Handling

Java uses exception handling to manage runtime errors.

try {
int a = 10 / 0;
} catch (Exception e) {
System.out.println("Error occurred");
}
