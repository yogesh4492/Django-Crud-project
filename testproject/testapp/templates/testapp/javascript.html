JavaScript Programming Language

A high-level, interpreted programming language that makes web pages interactive and dynamic. One of the core technologies of the web alongside HTML and CSS.

History of JavaScript

JavaScript was created in 1995 by Brendan Eich while working at Netscape. It was originally developed in just 10 days and was first named Mocha, then LiveScript, and finally JavaScript.

Despite the name, JavaScript is not related to Java. In 1997, it was standardized as ECMAScript (ES).

How JavaScript Works

JavaScript code is executed by a JavaScript engine inside the browser (like V8 in Chrome). The engine reads the code, interprets it, and executes it line by line.

JavaScript can:

  • Manipulate HTML elements
  • Change CSS styles
  • Handle user events (clicks, input)
  • Communicate with servers

Basic JavaScript Syntax

let message = "Hello, JavaScript";
console.log(message);

JavaScript is case-sensitive and uses statements, variables, functions, and objects.

JavaScript uses three keywords to declare variables:

let
const

Functions

Functions are reusable blocks of code that help in writing clean and maintainable code.

Events in JavaScript

JavaScript responds to user actions using events such as:

DOM (Document Object Model) allows JavaScript to interact with HTML elements.
